Contribute

There are several ways to contribute to SolAR framework.

You can :

  • contribute to Core framework, by defining API, architecture, framework tools …​ this typically needs software engineering and architecture skills.

  • contribute to Components, by creating a new SolAR component, that can be used in a pose estimation solution…​ this typically needs computer vision skills (and software skills).

Please feel free to contact us !

Create a component

Anyone can create a new component. C++ and computer vision skills are useful.

You can create a component :

  • for your own usage, because you need a new component for your pose estimation scheme. You can keep it private.

  • for the comunity, as you think your component can be useful for other developers. You can give it public in the SolAR libraries.

Please contribute !
