how

SolAR framework is Open-source

his framework is open-source, under Apache v2 licence, that lets you use it for research as well as commercial purposes adressing various domains (smart factory, smart home, real estate, e-health, etc).

Nevertheless, the components you will connect to develop your own solution can be under various licences. Always check the compatibility of the components licences, the SolAR community can not be held reponsible for a violation of the licenses of the components that you use.

SolAR defines component interfaces

SolAR framework is a solution to assemble vision components in order to create a pose estimation solution.

Core SolAR is the solution to assemble vision components

SolAR Core is the solution implemented to make possible to connect components in a pose estimation solution.

SolAR components are provided by SolAR framework

SolAR components are vision components "SolAR compatible" that can be used in the SolAR framework.

SolAR compatible means that your component must follow SolAR API defined in SolAR Core.

SolAR can use third parties

SolAR helps you to use vision components in order to create pose estimation.

SolAR can use third parties, as some well-known opensource vision frameworks.

For instance, SolAR has already some components based on OpenCV.

As so, you can benefit from the best vision components, and assemble them.
