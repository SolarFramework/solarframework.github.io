SolAR roadmap

The SolAR source code is going to be released for Windows and Linux in early November. It will include the interface definition required to build simple camera pose estimation pipelines (samples will be given for natural image markers and fiducial markers) as well as a Unity plugin allowing AR service developers to create their first AR applications using SolAR.

The following roadmap presents the features that will be implemented in a near future on SolAR core (pipeline, multi-threading, openVX compliance, Android and iOS support, etc.) as well as the new expected components allowing the creation of extended camera pose estimation pipelines (SLAM, visual/inertial, depth sensor support, etc.).

This roadmap is a projection, and may be subject to modifications. Please contact us if you have any questions.

All contributions are welcome to bring about this roadmap!
