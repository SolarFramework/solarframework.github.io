create the SolAR installer

this installs binaries and samples, as indicated in the "use it" part of the <SolAR> website.

You have to :

  1. install inno setup : http://www.jrsoftware.org/isdl.php#stable

  2. create a directory for your setup. Example called "setup".

  3. please get the iss file to create the Solar installer there : https://raw.githubusercontent.com/SolarFramework/binaries/master/inno-setup-SolarLibraries.iss

  4. update what will be installed for SolAR : # SolAR Libraries # third parties # SolAR Samples

Update Solar libraries

Versionning

First, please take care of versionning the code corresponding to your installer:

  1. please tag the source code

  2. please replace the version in all cmakelist.txt and/or the .pro files of the SolAR libraries with the good version

TARGET = SolARFramework INSTALLSUBDIR = bcomBuild FRAMEWORK = $$TARGET VERSION=G.R.C

  1. please also replace the version of the library you need in the packagedependencies.txt files before compiling

SolARFramework|G.R.C|SolARFramework|bcomBuild|url_repo_artifactory

Compile

Secondly, update, compile and store binaries in a directory called "installeurs" under your "setup directory".

please note that installers are based on master or develop branch. This means a merge workd, validation of code muste be done before generating the installer.

You can test the installer creation thanks to the content here : \\filer.b-com.local\share\common\PROJECTS\ARGO\atadrist\SolARSETUP image::images/installeurcontent.png[https://b-com.com/en]
