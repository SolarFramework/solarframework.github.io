Please contribute

SolAR framework is Open-source

This framework is open-source, under Apache v2 licence, that lets you use it for research as well as commercial purposes adressing various domains (smart factory, smart home, real estate, e-health, etc).

Nevertheless, the components you will connect to develop your own solution can be under various licences. Always check the compatibility of the components licences, the SolAR community can not be held reponsible for a violation of the licenses of the components that you use.

How to use

SOlAR framework can be used by :

  • pose estimation developers (using a C++ API) and

  • designers (using an editor),

to create a good pose estimation solution.

It can be used by the developer of AR application in their IDE.
